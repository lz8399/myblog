##### How to enable "Forward Rendering"

<!--more-->

Project Settings -> Rendering -> "Forward Shading" Checkbox (now you can use that silky MSAA)
{{< alert success >}}
Default Rendering Mode is Deferred Shading.
{{< /alert >}}
